# QR Verifier

Прогрессивное веб-приложение для верификации QR кодов с поддержкой офлайн режима.

## Возможности

- 📱 **Сканирование QR кодов** с помощью камеры устройства
- 💾 **Локальная база данных** на основе IndexedDB (Dexie)
- 🔄 **Автоматическая синхронизация** с сервером
- 📴 **Офлайн режим** с кэшированием через Service Worker
- 📊 **История сканирований** с детальной информацией
- 🎨 **Современный UI** с адаптивным дизайном
- 🚀 **PWA** - возможность установки как нативное приложение

## Технологии

- **Frontend**: React 19 + TypeScript + Vite
- **Database**: Dexie (IndexedDB wrapper)
- **QR Scanner**: html5-qrcode
- **PWA**: Workbox + vite-plugin-pwa
- **Styling**: CSS3 с градиентами и анимациями
- **Icons**: Lucide React

## Установка и запуск

1. **Клонируйте репозиторий**
   ```bash
   git clone <repository-url>
   cd qr-verifier
   ```

2. **Установите зависимости**
   ```bash
   npm install
   ```

3. **Настройте переменные окружения**
   ```bash
   cp .env.example .env
   # Отредактируйте .env файл согласно вашим настройкам
   ```

4. **Запустите в режиме разработки**
   ```bash
   npm run dev
   ```

5. **Соберите для продакшена**
   ```bash
   npm run build
   ```

## Архитектура

### Локальная база данных

Приложение использует IndexedDB через библиотеку Dexie для хранения:
- **QR Records** - результаты сканирования
- **Sync Status** - информация о синхронизации

### Синхронизация

Сервис синхронизации автоматически:
- Отправляет новые записи на сервер (когда доступен интернет)
- Получает обновления с сервера
- Управляет состоянием синхронизации

### PWA функции

- Service Worker для кэширования ресурсов
- Офлайн поддержка для всех основных функций
- Манифест для установки приложения
- Автоматические обновления

## API сервера

Приложение ожидает REST API со следующими эндпоинтами:

```
POST /api/qr-records     # Создание новой записи
GET /api/qr-records      # Получение записей
GET /api/qr-records?since=<timestamp>  # Получение записей после даты
```

### Формат данных

```typescript
interface QRRecord {
  id?: number;
  qrCode: string;
  isValid: boolean;
  timestamp: Date;
  data?: Record<string, unknown>;
  synced: boolean;
  serverRecordId?: string;
}
```

## Настройка валидации

В файле `src/App.tsx` в функции `validateQRCode()` можно настроить логику валидации QR кодов под ваши требования.
